# Juego de Lucha - Patrón Decorator

## Descripción
Este proyecto implementa un juego de lucha en Java utilizando el patrón estructural Decorator.

## Funcionalidades
- Ataques básicos
- Espada con probabilidad de daño extra
- Escudo con reducción de daño probabilística
- Superpoder cada 3 turnos:
  - Curación
  - Daño extra
  - Stun

## Patrón utilizado
Decorator
